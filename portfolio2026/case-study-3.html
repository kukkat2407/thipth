Case Study · Real Estate · Batdongsan.com.vn

Map view - Bring new experiences to home seekers

Overview

Project Map view (Bản đồ nhà đất)
Users Consumer - Buyer, Renter, Investor
My role Product Designer - owned end-to-end from design solution to gathering user feedback after launch
Stakeholders Head of Product, Product Manager, UX Researcher, Data Analyst, Engineering Manager, Developer, QA
Timeline Jun – Dec 2024
Impact Increased CTR from SRP to LDP per session · User experience rating 4.2★

Problem statement

One of the main consumer journeys on Batdongsan.com.vn is searching apartments for sale and for rent. After synthesising several user research sessions and interviews, we found that nearly 80% of users regularly go to Google Maps to find information about location, nearby amenities, and routes. Google Maps is a powerful tool users rely on in the early searching stage - alongside Batdongsan.com.vn and Facebook.

Moreover, in the Vietnamese marketplace there is no powerful prop-tech platform that leads map-based searching behaviour the way Zillow or Redfin do in the US. So, what if Batdongsan.com.vn offered this new experience?

We hypothesised that implementing Map view would increase the click-through rate (from SRP to LDP) because it allows users to navigate and explore listings in an area of interest more intuitively.

Design solution

The map view is a viewing mode for the Search Results Page (SRP) that works alongside the list view. Users can switch between views using the View Mode button.

The map view layout has two parts. The left side displays all the information and functions from the list view. The right side is the map zone, where users can interact with the map, property markers, search for listings, view nearby amenities, and more. This lets users view listing information while intuitively seeing its location on the map.

Users can also search listings using the Draw to Search freehand drawing tool. This feature lets users draw any area on the map where they want to search.

Many users love this tool because it supports flexible use cases - searching multiple locations or specific areas such as part of a street, a specific zone within a neighbourhood, or a combined area across multiple districts or wards. These use cases cannot be served by the regular search tool; only "Draw to Search" can do that.

Impact

  1. User rating (Survey run Feb 2025, n = 390)
  2. Compared to the List view, Map view increased click-through rate from SRP to LDP per session by ~2% on desktop web and ~1% on mobile web (Analysed Nov – Dec 2024)
  3. Larger adoption observed in Land for sale (~80% interacting) and Private house for sale (~77% interacting)
  4. Over 80% of users use the map in their early stage (before finding any property). Most use it to view property location, do market research, and gain price insight.
